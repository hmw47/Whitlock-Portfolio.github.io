p5js Landscape and Portrait

The goal of composing a landscape and a self-portrait using 2D primitives, colors, fill, and strokes on p5js was to practice our drawing skills using code.

We used the editor on the p5js site to create a self-portrait and a landscape, using both setup and draw functions.

Technical Notes

  • p5js

Portion of the Coding for the Drawings

The challenge with drawing on p5js for me was placing objects in the correct spot. I understand that I can find accurate coordinates using the size of the frame, but it would only get so far and I would end up having to tweak my numbers to get an exact location. It took me longer, having to do that, but I was still successful. It was refreshing to keep the shapes simple, and it made clean, fun compositions. Even though we had to use “primitive” techniques, I could still accomplish depth and light with layering and highlights. We were also allowed to use up to two emojis in the landscape, so I made that work a little more silly and playful.
