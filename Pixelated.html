Pixelated

Using Adobe Photoshop to create pixel art images, we created eight different works, two per dimension parameter. We explored creating “low quality”, pixelated images from both extracted fragments from larger images found on the web and using resized subject matter from larger contexts online.

I used the fixed-size selection tool and crop tool in Photoshop to extract small details from images to hone in on a new focal point for four of the images. I used the resize function in Photoshop to adjust the size of the images for the other portion of the project. I then used the index color function to experiment with color limitations.

Original Images

As the first active project in this course, saying it was a challenge is an understatement. I had to overcome the mental block of telling myself that I couldn’t figure it out because “I’m not tech savvy”. After the first submission, the only section of the rubric I got a perfect score on was my file organization for the submission, due to user error with the tools required for this project. I was able to revisit my work and resolve the issue by further exploring the software and its tools. I learned how to be more intentional with the tools I use to achieve a certain result and how important detail is when it comes to digital media. In the end, I got a perfect score on my “space” themed, pixelated project.
